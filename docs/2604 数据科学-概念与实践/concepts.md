## 纵向依赖链条 

第1章 → 第2章 → 第3章 → 第4-7章 → 第8章 → 第9-12章

第1章定义问题：监督/无监督决定后续算法选择

第2章给出流程：告诉我们“先特征选择→再建模→最后评估”

第3章提供原料：统计量和可视化帮我们理解数据长什么样，发现异常值

第4-7章是工具箱：根据问题类型选工具

预测**类别** → 第4章分类

预测**数值** → 第5章回归

找搭配**规则** → 第6章关联

自动**分组** → 第7章聚类

第8章是裁判：用混淆矩阵、ROC/AUC判定（评估）哪个模型更好

第9-12章是高级应用：在以上基础上加上文本、图像等特殊数据类型
